# Simple Inventory Program

A simple command-line inventory management program that stores items in a SQLite database. Users can create, view, edit, and remove items from the inventory.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Make sure Python is installed on your computer.
2. Open a terminal in the project folder.
3. Run the program with Python (for example, `py invintory_program.py` on Windows).

Instructions for using the software:

1. Option 1 from the main menu to add a new item.
2. Option 2 to view the inventory and select an item ID to edit or remove it.
3. Option 3 just exits the program.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Python 3 
* sqlite3 - (included with Python's standard library)
